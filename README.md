# Registration Form Website

A responsive and interactive Registration Form website built using **HTML, CSS, and JavaScript**. This project demonstrates front-end web development concepts such as form structure, styling, input validation, and user interaction.

## Features

* User-friendly registration form
* Responsive design for mobile and desktop
* Real-time form validation using JavaScript
* Error messages for invalid inputs
* Clean and modern UI design
* Password validation support
* Interactive submit functionality

## Technologies Used

* **HTML5** – Structure of the webpage
* **CSS3** – Styling and responsive layout
* **JavaScript** – Form validation and interactivity

## Purpose of the Project

The main purpose of this project is to improve front-end development skills by creating a fully functional registration form with validation features. It helps in understanding DOM manipulation, event handling, and responsive web design.

## Learning Outcomes

Through this project, I learned:

* Creating forms using HTML
* Designing responsive layouts with CSS
* Validating user input with JavaScript
* Handling events and DOM manipulation
* Improving user experience with interactive feedback
